Docodylus components respect `dark` and `light` themes in Docusaurus environments.

Docusaurus uses attributes for theming, rather than CSS classes, like some frameworks use. Enable theming in non-Docusaurus environments by setting attribute `data-theme` on the `html` element of your document. Supported values are `dark` and `light`
